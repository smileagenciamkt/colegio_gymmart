# 🏫 Colegio Gymmart - Sistema de Gestión de Asistencia

[![Version](https://img.shields.io/badge/version-1.1.0-blue.svg)](https://colegiogymmart.com)
[![Status](https://img.shields.io/badge/status-active-success.svg)](#)
[![Tech](https://img.shields.io/badge/tech-HTML5%20%7C%20CSS3%20%7C%20JS%20%7C%20Supabase-orange.svg)](#)

Un sistema de asistencia escolar moderno, intuitivo y eficiente diseñado para el **Colegio Gymmart**. Esta aplicación permite a los docentes y administradores gestionar la presencia del alumnado en tiempo real, con una estética de alta gama (estilo "churro") y funcionalidades avanzadas de reporte.

## 🚀 Características Pro
- **Panel de Control Dinámico**: Visualización de estadísticas en tiempo real con gráficos de barras integrados.
- **Nubecitas (Quick-tags)**: Acceso instantáneo a listas de alumnos ausentes y justificados.
- **Filtro de Historial**: Navegación por fechas pasadas para auditorías de asistencia.
- **Asistencia con un Click**: Interfaz rápida para marcar Presente, Ausente o Justificado.
- **Resúmenes para WhatsApp**: Generación automática de reportes profesionales listos para enviar por mensajería.
- **Importación Masiva**: Soporte para carga de alumnos vía Excel/CSV con mapeo inteligente de columnas.
- **Identidad Premium**: Interfaz basada en Glassmorphism con tipografía moderna (Outfit) y micro-animaciones.

## 🛠️ Stack Tecnológico
- **Frontend**: HTML5, CSS3 (Vanilla), JavaScript ES6+.
- **Backend-as-a-Service**: [Supabase](https://supabase.com) (Autenticación y Base de Datos en tiempo real).
- **Gráficos**: [Chart.js](https://www.chartjs.org/) para visualización de datos.
- **Procesamiento de Archivos**: [SheetJS (XLSX)](https://sheetjs.com/) para importación de datos.

## 📲 Instalación (PWA)
Esta aplicación está optimizada como **Progressive Web App (PWA)**. 
- **En Android**: Abre el sitio en Chrome, toca el menú de tres puntos y selecciona "Instalar aplicación".
- **En iOS**: Abre el sitio en Safari, toca el botón de "Compartir" y selecciona "Agregar a la pantalla de inicio".
